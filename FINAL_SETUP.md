---

### Step 2: Seed the Market Data
Since the market data crawler usually runs at 6:00 PM IST, the database starts empty. Run this command in your terminal to **manually trigger the first data scan**:

```powershell
# Run this inside the et-investor-intelligence folder
docker exec -it et-backend python -c "from backend.tasks.scheduled import daily_data_refresh, scan_opportunity_radar; daily_data_refresh(); scan_opportunity_radar()"
```

**This command will:**
1. Download today's NSE Bhavcopy (Prices).
2. Fetch SEBI Bulk/Block deals.
3. Run the AI "Opportunity Radar" to generate the first batch of signals.

### Step 3: Verify the Full Stack
1. **Frontend**: Open `http://localhost:3000` — You should see live Nifty values and the signals ticker moving.
2. **AI Chat**: Navigate to `/chat`, upload any sample portfolio PDF/CSV, and ask "How is my portfolio doing?"
3. **Daily Wrap**: Navigate to `/video` and click "Generate Nightly Wrap" to see the video engine build a market summary.
